# mechamoandrey.com

Meu portfolio. Construído para ser direto: stack real, projetos reais, sem enrolação.

## Stack

- **Next.js 15** (App Router)
- **TypeScript** (strict)
- **Tailwind CSS v4**
- **Framer Motion** — animações de entrada e transições
- **GSAP + ScrollTrigger** — scroll horizontal da seção de trabalhos
- **next-themes** — dark/light mode sem flash

## Dev

```bash
npm install
npm run dev
```

## Deploy

Vercel. Push para `main` sobe automaticamente.
